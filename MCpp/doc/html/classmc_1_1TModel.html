Detailed Description

template<typename T>
class mc::TModel< T >

mc::TModel is a C++ class for definition of Taylor model environment. Propagation of Taylor models for factorable functions is via the C++ class mc::TVar. The template parameter corresponds to the type used to propagate the remainder bound.
